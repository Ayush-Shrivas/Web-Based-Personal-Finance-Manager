# FinTrack  
## Web-Based Personal Finance Management System

FinTrack is a modern, secure, and user-friendly **personal finance management web application** built using **Python, Flask, and SQLite**.  
It allows users to track income and expenses, analyze savings, and manage personal finances through an intuitive web interface.

This project is designed with a strong focus on **clean architecture, modularity, data persistence, and scalability**, making it suitable for **academic submissions, internships, and portfolio presentation**.
